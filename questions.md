# Что такое регулярные выражения и в каких задачах обработки текста они наиболее эффективны?
* Регулярные выражения (RegExp) — это формальный язык поиска и осуществления маничпуляций с подстроками в тексте, основанный на использовании метасимволов. По сути, это шаблон (образец), который описывает набор строк.
* Они наиболее эффективны в задачах, где нужно работать с текстом, имеющим определенную структуру или формат.
# Какой модуль в Python необходимо импортировать для работы с регулярными выражениями?
* Модуль называется re, "import re"
# В чём разница между методами re.search() и re.findall()? Какой из них возвращает список всех совпадений?
* re.search(): Ищет первое же совпадение с шаблоном во всей строке. Если совпадение найдено, возвращает специальный объект Match
* re.findall(): Ищет все непересекающиеся совпадения с шаблоном во всей строке и возвращает их в виде списка (list) строк
* Список всех совпадений возвращает re.findall().
# Если вам нужно проверить, что строка начинается с определённого паттерна, какой метод модуля re вы будете использовать?
* Лучше всего использовать метод re.match(). Важное отличие re.match() от re.search(): match() проверяет совпадение строго с начала строки. Если строка начинается не с паттерна, match() вернет None.
# Что означает метасимвол . (точка) в регулярных выражениях?
* Точка (.) — это метасимвол, который обозначает один любой символ, кроме символа новой строки (\n)
# Что такое группировка в регулярных выражениях и какой синтаксис используется для её обозначения?
* Группировка — это объединение части регулярного выражения в единое целое.
* Синтаксис: Круглые скобки ( ).
